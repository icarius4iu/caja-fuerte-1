# Navbar

## Propósito

Es la barra de navegación principal. Contiene el logo de "Caja Fuerte", enlaces de navegación (Cursos, Calendario, Bonos, etc.), y un botón de llamada a la acción ("Empieza Gratis"). Incluye un botón para menú móvil (hamburguesa), aunque la lógica de apertura del menú no está incluida en este archivo (probablemente manejada externamente o pendiente de implementación).

## Props

Este componente no acepta props.
