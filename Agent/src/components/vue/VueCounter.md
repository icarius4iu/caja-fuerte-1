# VueCounter

## Propósito

Componente de contador interactivo implementado con Vue 3 (Composition API). Sirve como demostración de la integración de islas de Vue en Astro.

## Props

Utiliza `<slot />` para contenido hijo.

## Estado

- **count:** Estado reactivo (`ref(0)`).
- **Métodos:** `add` (+1), `subtract` (-1).
