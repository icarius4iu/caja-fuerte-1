# AdminHeader

## Propósito

Cabecera para el panel de administración. Contiene botón de menú (para móvil), barra de búsqueda (visual) y menú de perfil de usuario con avatar.

## Props

Este componente no acepta props.

## Estado

- **Interacción:** Los botones de búsqueda y perfil son visuales y no tienen lógica JS implementada en este archivo para desplegar menús o realizar búsquedas reales.

## Cambios Recientes

- **Fecha:** 2026-01-17
- **Descripción:** Documentación inicial creada.

## Dependencias

- Iconos SVG inline.
- Imagen de avatar externa (unsplash).
