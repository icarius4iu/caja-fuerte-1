# AdminSidebar

## Propósito

Barra lateral de navegación para el panel de administración. Muestra el logo y enlaces a las diferentes secciones (Dashboard, Content, Users, Settings).

## Props

Este componente no acepta props.

## Estado

- **Responsive:** Utiliza clases de Tailwind (`transform`, `-translate-x-full`, `md:translate-x-0`) para ocultarse en móviles y mostrarse en escritorio. La lógica para alternar su visibilidad en móvil (toggle) no está presente en este componente (probablemente dependa de un estado externo o script en el layout).
