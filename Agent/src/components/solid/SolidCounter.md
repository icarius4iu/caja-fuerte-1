# SolidCounter

## Propósito

Componente de contador interactivo implementado con SolidJS. Demostración de integración Solid en Astro.

## Props

| Prop       | Tipo          | Descripción                                          |
| ---------- | ------------- | ---------------------------------------------------- |
| `children` | `JSX.Element` | Contenido opcional a renderizar debajo del contador. |

## Estado

- **count:** Señal reactiva `createSignal(0)`.
- **Métodos:** `add`, `subtract`.
