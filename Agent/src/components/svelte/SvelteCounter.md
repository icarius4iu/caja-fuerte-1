# SvelteCounter

## Propósito

Componente de contador interactivo implementado con Svelte 5 (Runes). Demostración de integración Svelte en Astro.

## Props

| Prop       | Tipo      | Descripción                                                     |
| ---------- | --------- | --------------------------------------------------------------- |
| `children` | `Snippet` | Snippet de contenido opcional a renderizar debajo del contador. |

## Estado

- **count:** Estado reactivo con runas `let count = $state(0)`.
- **Métodos:** `add()`, `subtract()`.
