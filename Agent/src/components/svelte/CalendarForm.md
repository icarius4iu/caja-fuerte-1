# CalendarForm (Svelte)

## Propósito

Formulario para añadir nuevas promociones al calendario. Permite configurar título, categoría, fechas, beneficios e imagen.

## Props

Este componente no acepta props.

## Estado

- **Campos del formulario:** `title`, `category`, `startDate`, `endDate`, `benefit`, `imageUrl`.
- **CATEGORIES:** Array constante con categorías (apuesta_gratis, giros_gratis, supercuotas).
- **Métodos:** `handleSubmit()` simula el guardado de la promoción.
