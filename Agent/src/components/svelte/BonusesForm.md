# BonusesForm (Svelte)

## Propósito

Formulario para crear nuevos bonos en el panel de administración. Permite ingresar título, tipo, monto, código, fecha de expiración y descripción.

## Props

Este componente no acepta props.

## Estado

- **Campos del formulario:** `title`, `type`, `amount`, `code`, `expiryDate`, `description`.
- **BONUS_TYPES:** Array constante con los tipos de bonos disponibles (welcome, reload, cashback, etc.).
- **Métodos:** `handleSubmit()` valida y simula el guardado del bono (log en consola y alerta).
