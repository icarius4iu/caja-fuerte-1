# HeroSection

## Propósito

Presenta la sección principal (Hero) de la Landing Page. Muestra el título principal, subtítulos, botones de llamada a la acción ("Vengo por mi fija", "Ver Demo"), imágenes de prueba social (avatares de usuarios) y una gráfica visual con efecto de desenfoque.

## Props

Este componente no acepta props.

## Estado

Este componente es estático. Usa clases de Tailwind para animaciones CSS puras (`animate-fade-in-up`, `animate-bounce`).

## Cambios Recientes

- **Fecha:** 2026-01-17
- **Descripción:** Documentación inicial creada.

## Dependencias

- Font Awesome (iconos).
- Imágenes alojadas externamente (lh3.googleusercontent.com).
