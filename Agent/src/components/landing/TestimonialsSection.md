# TestimonialsSection

## Propósito

Muestra testimonios de usuarios satisfechos en un diseño de rejilla (grid) de 3 columnas. Cada tarjeta incluye la cita, nombre, rol y avatar del usuario.

## Props

Este componente no acepta props.

## Estado

Estático.

## Cambios Recientes

- **Fecha:** 2026-01-17
- **Descripción:** Documentación inicial creada.

## Dependencias

- Font Awesome (iconos de comillas).
- Imágenes de avatares externas.
