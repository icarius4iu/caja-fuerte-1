# SimulatorSection

## Propósito

Promociona la herramienta de simulación de apuestas. Muestra beneficios en una lista y una representación visual de la interfaz del simulador con gráficos y métricas estáticas simuladas.

## Props

Este componente no acepta props.

## Estado

Este componente es estático. Incluye gráficos SVG inline para la representación visual de datos.

## Cambios Recientes

- **Fecha:** 2026-01-17
- **Descripción:** Documentación inicial creada.

## Dependencias

- Font Awesome (iconos de check).
- SVG inline para gráficos.
