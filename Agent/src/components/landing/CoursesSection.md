# CoursesSection

## Propósito

Muestra un carrusel o lista horizontal de cursos disponibles (Fundamentos, Psicología, Bankroll, Excel). Permite desplazamiento horizontal (`snap-x`) para navegar entre las tarjetas de cursos.

## Props

Este componente no acepta props.

## Estado

- **Interacción:** El desplazamiento es nativo mediante CSS (`overflow-x-auto`, `snap-start`), no utiliza JavaScript para el comportamiento del carrusel en este archivo.
- **Botones:** Incluye botones de navegación (< >) visuales, pero sin lógica JS vinculada en este archivo.
