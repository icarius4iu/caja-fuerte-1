# FeaturesSection

## Propósito

Sección que destaca las características comunitarias y beneficios ("Comunidad Exclusiva"). Utiliza un diseño vibrante con gradientes, efectos de desenfoque y un elemento visual interactivo simulado de chat de Discord.

## Props

Este componente no acepta props.

## Estado

Este componente es principalmente estático. Utiliza transformaciones CSS (`hover:rotate-0`) para crear interactividad visual al pasar el mouse sobre la tarjeta del chat.

## Cambios Recientes

- **Fecha:** 2026-01-17
- **Descripción:** Documentación inicial creada.

## Dependencias

- Font Awesome (iconos).
- Clases de utilidad personalizadas o arbitrarias de Tailwind (ej. `perspective-1000`).
