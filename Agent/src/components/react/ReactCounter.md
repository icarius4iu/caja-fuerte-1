# ReactCounter

## Propósito

Componente de contador interactivo implementado con React. Sirve como demostración de la integración de islas de React en Astro.

## Props

| Prop       | Tipo        | Descripción                                          |
| ---------- | ----------- | ---------------------------------------------------- |
| `children` | `ReactNode` | Contenido opcional a renderizar debajo del contador. |

## Estado

- **count:** Estado numérico interno gestionado con `useState(0)`.
- **Métodos:** `add` (+1), `subtract` (-1).
