# PreactCounter

## Propósito

Componente de contador interactivo implementado con Preact. Demostración de integración Preact en Astro.

## Props

| Prop       | Tipo                | Descripción                                          |
| ---------- | ------------------- | ---------------------------------------------------- |
| `children` | `ComponentChildren` | Contenido opcional a renderizar debajo del contador. |

## Estado

- **count:** Estado numérico interno `useState(0)`.
- **Métodos:** `add`, `subtract`.
